# imageGenerator
🔗[openAI](https://platform.openai.com/) api 활용, 프롬프트 및 이미지 생성 서비스

<br>

## 사용 기술
JavaScript, HTML, CSS

<br>

## 활용 api 
- Text Completion API(gpt-3.5-turbo model)
- Images API (DALL·E model)

<br>
